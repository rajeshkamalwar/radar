# RADAR SaaS

**R.A.D.A.R.** means **Reputation, Authority, Discovery, Analytics & Response**.

This is the first product scaffold for RADAR: an enterprise brand visibility intelligence OS with separate specialist dashboards for search, AI visibility, reputation, reviews, local discovery, social, influencers, trends, authority, competitors, funnel prediction, and executive command.

## Current Scope

- Executive RADAR dashboard
- Module grid for all RADAR systems
- Cross-signal intelligence panel
- Work ledger
- Approval queue
- Next-best actions
- Static data model ready to connect to APIs

## Run Locally

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```
